## 搬瓦工 BWG 是什么，为什么这么多人用

BWG 是加拿大 IT7 Networks Inc. 旗下品牌，2004 年成立，算是老牌 VPS 商家了。之所以被中国用户亲切地叫做"搬瓦工"，是因为它的英文拼写和汉语拼音"banwagong"长得太像了。

它有几个比较明显的特点：

自研了一套叫 **KiwiVM** 的控制面板，你可以在里面一键重装系统、查流量、甚至一键迁移到另一个机房，操作流畅。机房遍布美国洛杉矶、日本大阪/东京、中国香港、荷兰阿姆斯特丹等地，买一个套餐可以在多个机房自由切换。

支持**支付宝**付款——这对国内用户来说太友好了，不用搞什么国际信用卡。

整体定位：价格在 VPS 圈子里不算最便宜，但稳定性和线路质量打了很好的招牌，尤其是 CN2 GIA 线路在国内用户里口碑很好。
